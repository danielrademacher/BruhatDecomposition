4.1-1 LGOStandardGensSp
‣ LGOStandardGensSp( i, alpha )( function )

Returns: true

Let \(\alpha \in GF(p^f)\), \(\alpha = \sum a_l \omega^l\), \(\omega\) a primitive element. Suppose further that Tipos is a list of transvections of the form \({ t_{i,i-1}(\omega^{ell}) }\),\( 2 \leq i \leq d\), \(0 \leq ell \leq \neq f\). Then this function computes \(t_{i,i-1}( \alpha )\) by (Lemma 4.2) and saves the result in tvpos.
